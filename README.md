 # TMPFS Maker <img src="https://em-content.zobj.net/source/microsoft-teams/363/face-exhaling_1f62e-200d-1f4a8.png" width="28">

A tiny, playful C++ tool that creates simple TMPFS mounts—because sometimes you just want a RAM-backed filesystem without all the drama.

## What it does
- Creates a minimal TMPFS mount
- Stays lightweight (about 200 lines of code)
- Exists mostly for fun, curiosity, and messing around with system internals

## Why?
Because writing a small utility in C++ that pokes the OS is oddly satisfying.  
Also because TMPFS is cool and nobody can convince me otherwise.

## Notes
This is not meant for production.  
It’s just a compact example project you can read, break, or improve however you like.
